# Clase_Git
Clase de Git y GitHub 🚀
Bienvenido/a a este repositorio de práctica.
Aquí aprenderemos el flujo de trabajo básico con Git y GitHub de forma colaborativa.

## ¿Cómo participar?

- Haz Fork de este repositorio (botón arriba a la derecha)

- Clona tu fork en tu ordenador

- Crea una rama con tu nombre: git checkout -b tu-nombre

- Añade tu línea en la sección Participantes de abajo

- Haz git add, commit y push de tus cambios

- Abre un Pull Request hacia este repositorio


Espacio para Coders:
